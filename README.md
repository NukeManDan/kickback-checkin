# Kickback Auto Check-In Web Interface

live demo: https://nukemandan.github.io/kickback-checkin/


## Develop:

Install:
```bash
npm install
```

Run Dev Server:
```bash
npm run dev
```
Build Static Site:
```bash
npm run build
```

_note: build is configured to go to `docs` folder in `quasar.conf.js` for github pages_
